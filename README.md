音频可视化播放器

注意：本播放器使用AI生成

这是一个功能丰富的现代化音频播放器，具有音频可视化、歌词同步、粒子背景和智能歌单管理等高级功能。

1、	主要特性

（1）音频功能
- 🎵 支持多种音频格式：MP3、FLAC、OGG、WAV、AAC、M4A
- 🎚️ 音频可视化：实时音频频谱分析 
- 🔄 播放模式：顺序播放、随机播放、循环播放
- ⏯️ 媒体控制：支持系统媒体按键和 MediaSession API

（2）可视化效果
- 🌈 动态粒子背景：音频驱动的粒子动画系统
- 🎨 智能色彩提取：从专辑封面提取主色调并应用到界面
- 🔄 专辑封面动画：根据音频节奏进行旋转和缩放
- 🌫️ 背景模糊：基于专辑封面的动态模糊背景
- 📊 频谱可视化：可调节的频率响应曲线和渐变效果

（3）歌词系统
LRC歌词支持：标准LRC格式歌词同步显示
智能歌词匹配：自动匹配歌词文件与音频文件
歌词管理面板：批量导入、搜索和管理歌词文件
歌词高亮：实时同步的歌词滚动和高亮效果
仅支持MP3格式音频文件的内嵌歌词、其他文件需手动导入歌词文件

 歌单管理
- 📁 歌单系统：创建、编辑、删除自定义歌单
- 📋 拖拽排序：在播放列表中拖拽调整歌曲顺序
- 📤 导入/导出：歌单数据的保存与读取
- 🖱️ 右键菜单：快速将歌曲添加到歌单

 用户界面
可隐藏面板：播放列表和歌词面板可折叠
控制栏锁定：防止控制栏自动隐藏
拖放支持：直接拖放音频文件到播放器

 设置系统
详尽设置选项：覆盖播放、可视化、背景等各个方面
设置导入/导出：保存和恢复个性化配置
一键重置：快速恢复默认设置

2、	使用说明
 （1）播放控制
- 播放/暂停：点击播放按钮
- 上一曲/下一曲：使用控制栏按钮或快捷键
- 进度控制：点击进度条或拖动滑块
- 音量控制：使用音量滑块或系统音量

 （2）界面操作
- 隐藏播放列表：点击播放列表标题栏的箭头按钮
- 打开设置：点击控制栏的设置按钮
- 锁定控制栏：点击锁形图标

（3）文件添加与删除
- 单个文件：点击"添加音乐文件"按钮，可以添加音频文件
- 整个文件夹：点击"添加音乐文件夹"按钮，添加音乐文件夹支持读取其中的音频文件、设置文件与歌词文件
- 拖放文件：直接将音频文件、设置文件、歌词文件拖放到播放器界面
- 删除歌曲：歌曲最右侧有删除按钮
- 添加到歌单：点击播放列表项上的加号按钮
- 重新排序：拖拽播放列表项调整顺序
- 歌单文件需要在歌单面板单独导入

（4）设置列表

 播放设置
- 随机播放开关
- 循环播放开关
- 音频可视化开关
- 控制栏锁定开关
- 音量调节

 专辑封面设置
- 旋转速度控制
- 封面大小调整
- 联动动画开关
- 缩放方向设置
- 鼓声峰值阈值
- 封面变化程度
- 封面图片大小

 背景设置
- 背景亮度调节
- 背景模糊度控制

 可视化效果设置
- 渐变强度调节
- 响应曲线调整
- 平滑度控制

 颜色设置
- 饱和度调节
- 透明度控制
- 亮度调节

 歌词匹配设置
- 最低匹配程度设置
- 播放时自动匹配歌词开关

 粒子效果设置
- 粒子效果开关
- 粒子数量调节
- 粒子大小控制
- 粒子透明度设置
- 粒子响应度调节

3、	注意事项
- 歌曲内嵌封面只支持MP3与flac文件
- 歌曲匹配歌词文件是通过与歌词文件名匹配的，匹配不到可以适当调整最低匹配程度设置。
- 导入大量文件时可能需要较长的加载时间。
- 过多的粒子数量可能影响低端设备的性能。
