# 音频可视化播放器

这是一个功能丰富的现代化音频播放器，具有音频可视化、歌词同步、粒子背景和智能歌单管理等高级功能。

注意这个播放器使用AI进行编写

## 🌟 主要特性

### 音频功能
- 🎵 **支持多种音频格式**：MP3、FLAC、OGG、WAV、AAC、M4A
- 🎚️ **音频可视化**：实时音频频谱分析
- 🔄 **播放模式**：顺序播放、随机播放、循环播放
- ⏯️ **媒体控制**：支持系统媒体按键和 MediaSession API

### 可视化效果
- 🌈 **动态粒子背景**：音频驱动的粒子动画系统
- 🎨 **智能色彩提取**：从专辑封面提取主色调并应用到界面
- 📊 **频谱可视化**：可调节的频率响应曲线和渐变效果

### 歌词系统
- 📝 **LRC歌词支持**：支持读取MP3文件的内嵌歌词
- 🤖 **智能歌词匹配**：自动匹配歌词文件与音频文件
- 🔍 **歌词管理面板**：批量导入、搜索和管理歌词文件
- 🎯 **歌词高亮**：实时同步的歌词滚动和高亮效果

### 歌单管理
- 📁 **歌单系统**：创建、编辑、删除自定义歌单
- 📋 **拖拽排序**：在播放列表中拖拽调整歌曲顺序
- 📤 **导入/导出**：歌单数据的保存和分享功能
- 🖱️ **右键菜单**：快速将歌曲添加到歌单

### 用户界面
-  **可隐藏面板**：播放列表和歌词面板可折叠
-  **控制栏锁定**：防止控制栏自动隐藏
-  **拖放支持**：直接拖放音频文件、歌词文件、设置文件到播放器
-  **大封面**：有隐藏的上一首、播放、下一首按钮

### 设置系统
- ⚙️ **详尽设置选项**：覆盖播放、可视化、背景等各个方面
- 💾 **设置导入/导出**：保存和恢复个性化配置
- 🔄 **一键重置**：快速恢复默认设置

## 🚀 快速开始

### 直接使用
可直接打开html文件在浏览器中运行

### 导入音乐
- **单个文件**：点击"添加音乐文件"按钮
- **整个文件夹**：点击"添加音乐文件夹"按钮
- **拖放文件**：直接将音频文件拖放到播放器界面

## 基本操作

### 播放控制
- **播放/暂停**：点击播放按钮或按空格键
- **上一曲/下一曲**：使用控制栏按钮或快捷键
- **进度控制**：点击进度条或拖动滑块
- **音量控制**：使用音量滑块或系统音量

### 界面操作
- **隐藏播放列表**：点击播放列表标题栏的箭头按钮
- **展开歌词面板**：在平板/手机上自动适应布局
- **打开设置**：点击控制栏的设置按钮
- **锁定控制栏**：点击锁形图标

### 文件管理
- **删除歌曲**：将鼠标悬停在播放列表项上，点击出现的删除按钮
- **添加到歌单**：点击播放列表项上的加号按钮或使用右键菜单
- **重新排序**：拖拽播放列表项调整顺序
- **导入文件夹**：在播放列表点击导入文件夹，可以自动扫描并导入文件夹中的歌词文件、音频文件、设置文件。

## 注意事项

- 导入大量音频文件可能需要较长的加载时间（可以一开始选择一些歌曲少的歌单后再导入文件，减少要加载的歌曲封面，加快加载速度）
- 歌词文件需要与音频文件有匹配的文件名（例如歌词文件名使用歌名来命名，或歌词文件采用与歌曲文件相同的名称）
- 过多的粒子数量可能会卡顿
- 导入歌单只能在歌单界面导入
